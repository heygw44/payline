## 1. 전체 레이아웃 구조

### AppLayout.vue

모든 인증된 페이지가 공유하는 마스터 레이아웃입니다.

```
+------------------------------------------------------------------+
|  [AppHeader]                                                      |
|  +--------------------------------------------------------------+|
|  | 🧾 PayLine          |  김정산님 (ADMIN)  |  [로그아웃]        ||
|  +--------------------------------------------------------------+|
+--+-----------------------------------------------------------+---+
|  |                                                           |   |
|  | [AppSidebar]     [<router-view />]                        |   |
|  |                                                           |   |
|  | 📊 대시보드       +-----------------------------------+   |   |
|  | 🏪 업주 관리      |                                   |   |   |
|  | 📦 주문 관리      |         메인 콘텐츠 영역           |   |   |
|  | 💰 보상금 관리    |                                   |   |   |
|  | 📋 지급 관리      |                                   |   |   |
|  | ──────────       |                                   |   |   |
|  | 👤 회원 관리 *    |                                   |   |   |
|  |   (* ADMIN만)    +-----------------------------------+   |   |
|  |                                                           |   |
+--+-----------------------------------------------------------+---+
```

#### AppHeader 컴포넌트

| 요소 | 위치 | 동작 |
|------|------|------|
| 로고 + 프로젝트명 | 좌측 | 클릭 시 대시보드로 이동 |
| 사용자 이름 + 역할 | 우측 | authStore에서 가져옴 |
| 로그아웃 버튼 | 우측 끝 | POST /api/logout → 로그인 페이지 |

#### AppSidebar 컴포넌트

| 메뉴 항목 | 라우트 | 권한 | 아이콘 |
|-----------|--------|------|--------|
| 대시보드 | `/` | ALL | 📊 |
| 업주 관리 | `/owners` | ALL | 🏪 |
| 주문 관리 | `/orders` | ALL | 📦 |
| 보상금 관리 | `/rewards` | ALL | 💰 |
| 지급 관리 | `/settles` | ALL | 📋 |
| 회원 관리 | `/admin/members` | ADMIN | 👤 |

## 9. 공통 컴포넌트 명세

### 9.1 AppTable

| Props | 타입 | 설명 |
|-------|------|------|
| `columns` | `Array<{key, label, width?, align?, formatter?}>` | 컬럼 정의 |
| `data` | `Array<Object>` | 테이블 데이터 |
| `loading` | `Boolean` | 로딩 상태 |
| `emptyText` | `String` | 데이터 없을 때 메시지 |
| `@row-click` | `Event` | 행 클릭 이벤트 |

### 9.2 AppPagination

| Props | 타입 | 설명 |
|-------|------|------|
| `page` | `Number` | 현재 페이지 (0-based) |
| `totalPages` | `Number` | 전체 페이지 수 |
| `@change` | `Event(page)` | 페이지 변경 이벤트 |

### 9.3 AppModal

| Props | 타입 | 설명 |
|-------|------|------|
| `visible` | `Boolean` | 모달 표시 여부 (v-model) |
| `title` | `String` | 모달 제목 |
| `width` | `String` | 모달 너비 (기본 500px) |
| `@close` | `Event` | 닫기 이벤트 |

### 9.4 AppSearchBar

| Props | 타입 | 설명 |
|-------|------|------|
| `filters` | `Array<{key, label, type}>` | 필터 정의 (text/select/date) |
| `@search` | `Event(conditions)` | 검색 실행 이벤트 |
| `@reset` | `Event` | 초기화 이벤트 |

### 9.5 StatusBadge

| Props | 타입 | 설명 |
|-------|------|------|
| `status` | `String` | 상태 값 (PENDING, COMPLETED 등) |
| `type` | `String` | 배지 유형 (settle/order) |

### 9.6 AmountDisplay

| Props | 타입 | 설명 |
|-------|------|------|
| `amount` | `Number` | 금액 |
| `sign` | `Boolean` | +/- 부호 표시 여부 |

### 9.7 ConfirmDialog

| Props | 타입 | 설명 |
|-------|------|------|
| `visible` | `Boolean` | 표시 여부 (v-model) |
| `message` | `String` | 확인 메시지 |
| `confirmText` | `String` | 확인 버튼 텍스트 |
| `dangerous` | `Boolean` | 위험 동작 여부 (빨간 버튼) |
| `@confirm` | `Event` | 확인 이벤트 |
